# Welcome

Compass is an **AI workspace** for local folders. It is not a code-only IDE — you can write, fix, and organize notes, documents, data, and code with AI.

## What can you do?

1. Open a folder
2. Edit files
3. Ask AI questions / request edits
4. Preview proposals and apply them

## First 5 minutes

1. [Install](install.md) and launch
2. Set an API key (or Ollama) in [AI provider setup](ai-provider.md)
3. [Open a folder](open-project.md)
4. Ask something in the right-hand chat → [AI chat](../ai/chat.md)

## UI layout

| Area | Role |
|------|------|
| Left | File tree |
| Center | Editor (tabs) |
| Right | AI chat (Ask / Edit / Agent) |
| Bottom | Terminal (when shown) |
