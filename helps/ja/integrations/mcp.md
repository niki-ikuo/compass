# MCP

## 現状

**未対応です。** MCP（Model Context Protocol）連携は仕様上の後続項目です。

## いま近いもの

Compass 内蔵の [Agent](../ai/agent.md) が、ワークスペース内の読取・変更提案・制限付き実行を担います。外部 MCP サーバーの追加・接続 UI はありません。

## このページの扱い

対応したら、サーバーの追加方法・許可範囲・トラブルシュートをここに書きます。
