# はじめに

Compass は、ローカルフォルダ向けの **AI ワークスペース**です。コード専用 IDE ではなく、メモ・文書・データ・コードなどテキスト全般を AI と一緒に書く・直す・整理できます。

## 何ができる？

1. フォルダを開く
2. ファイルを編集する
3. AI に質問する / 変更を提案させる
4. 提案をプレビューして適用する

## 最初の5分

1. [インストール](install.md)して起動する
2. [AIプロバイダー](ai-provider.md)で API Key（または Ollama）を設定する
3. [フォルダを開く](open-project.md)
4. 右のチャットで質問してみる → [AIチャット](../ai/chat.md)

## 画面の見方

| 場所 | 役割 |
|------|------|
| 左 | ファイルツリー |
| 中央 | エディタ（タブ） |
| 右 | AIチャット（Ask / Edit / Agent） |
| 下 | ターミナル（表示時） |
